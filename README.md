# Number Guessing Game | C

## Description
This project is a simple console-based number guessing game developed using the C programming language.
The program generates a random number and allows the user to guess it until the correct number is found.

Loops and conditional statements are used to control the game flow and provide hints to the user.

## Features
- Random number generation
- User interaction through console
- Hint messages (too high / too low)
- Counts number of attempts
- Simple and beginner-friendly
